### 数据模型分配简介
```js
models
 |- betCenter.js // 处理有关投注界面数据
 |- dataTable.js // 处理报表管理数据如页数，当前页面，过滤数据
 |- form.js // 处理表格填写以及报错数据
 |- gameInfos.js // 处理彩种信息和历史
 |- helpCenter.js // 处理帮助信息页面信息
 |- instructions.js // 处理玩法说明文件
 |- layout.js // 处理页面展示
 |- order.js // 处理订单信息
 |- promotions.js // 处理优惠活动信息
 |- team.js // 处理下级用户信息
 |- transaction.js // 处理用户转账历史
 |- transfer.js // 处理用户充值提款
 |- trend.js // 处理走势图
 |- user.js // 处理用户信息
```
